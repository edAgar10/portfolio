Projects

Pizza Business Incremental

Incremental web game currently in development based on running a pizza delivery service. This project stemmed from developing a solar system model with orbits and lighting using Javascript and OpenGL shaders, with the idea of spreading the business across different planets. The main game consists of producing and managing different pizza ingredients so that you can fulfil orders, make money and buy upgrades. The project is built with scalability in mind, so that new ingredients and building types can be added as necessary. Currently working on the order system and allowing building placement on planets.

Rubiks Cube

Simple Rubik's Cube simulation as shown on the home page. While a smaller project, this helped improve my knowledge of the OpenGL rendering pipeline through use of shaders and post-processing, as well as general 3D maths. Working on improving controls to allow for different rotations as well as adding rotation animations. While mostly developed for visuals, potential for automatic solving/shuffle in future development.

Island World Generator Prototype

Work in progress prototype for a pirate game based on classic roguelikes made in Godot. Generates a world map with different islands using perlin noise, with the player sailing between different islands. Also includes a 2D grid based movement system for walking off the ship, as well as A* pathfinding around obstacles and solid blocks. Currently working on turn based combat as well as building on the island generation with techniques like wave function collapse. Also considering transferring the code base to a 3D environment for use in a different project.

Machine Learning Image Generator

Dissertation project to make a generative adversarial network that can take a dataset of pictures and output generated images. The pictures shown use a dataset of abstract and classical artwork, however a tagged database could be used to make more consistent imagery. Written in Python using Pytorch for machine learning and NumPy for graph plotting.
